# Porquinho Brasfeed

Demo de jogo 2D desenvolvido em Python com Pygame.

## Objetivo

O jogador controla um porquinho que deve escolher produtos Brasfeed para crescer.

- Produto Brasfeed: faz o porquinho crescer.
- Produto concorrente: deixa o porquinho doente.
- 5 produtos Brasfeed: vitória.
- 3 produtos concorrentes: derrota.
